# Как запустить приложение

## Способ 1: Через npm script (рекомендуется)

Откройте терминал в директории проекта и выполните:

```bash
cd "/Users/pskripilev/Downloads/Мои файлы/healt"
npm start
```

Или напрямую через expo:

```bash
npx expo start
```

## Способ 2: Через локальный бинарный файл

```bash
cd "/Users/pskripilev/Downloads/Мои файлы/healt"
./node_modules/.bin/expo start
```

## После запуска

Вы увидите QR-код и меню с командами:

- Нажмите `a` - для запуска на Android эмуляторе
- Нажмите `i` - для запуска на iOS симуляторе  
- Отсканируйте QR-код - для запуска на реальном устройстве через Expo Go

## Первый запуск приложения

1. Откройте приложение
2. Перейдите на вкладку "Настройки" (⚙️)
3. Нажмите кнопку "Сгенерировать начальные данные (7 дней)"
4. Включите переключатель "Включить симуляцию"
5. Выберите профиль пользователя (по умолчанию "Активный")

## Если возникают ошибки

### Ошибка с правами доступа npm

Попробуйте использовать полный путь к node:

```bash
/usr/local/bin/node node_modules/.bin/expo start
```

### Порт занят

Если порт 8081 занят:

```bash
npx expo start --port 8082
```

### Очистка кеша

Если приложение не запускается:

```bash
npx expo start --clear
```

## Требования

- Node.js версии 16 или выше
- Установленные зависимости (npm install уже выполнен)
- Для запуска на устройстве: приложение Expo Go из App Store / Google Play
