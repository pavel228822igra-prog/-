# Быстрый старт

## Запуск приложения

### Вариант 1: На реальном устройстве (РЕКОМЕНДУЕТСЯ)

1. Установите **Expo Go** на телефон:
   - Android: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)
   - iOS: [App Store](https://apps.apple.com/app/expo-go/id982107779)

2. Убедитесь, что телефон и компьютер в одной Wi-Fi сети

3. Запустите сервер:
   ```bash
   npm start
   ```

4. Отсканируйте QR-код:
   - **Android**: Откройте Expo Go → Нажмите "Scan QR code"
   - **iOS**: Откройте приложение "Камера" → Наведите на QR-код → Нажмите на уведомление

### Вариант 2: Веб-версия

1. Запустите сервер: `npm start`
2. В терминале нажмите `w`
3. Приложение откроется в браузере

### Вариант 3: iOS симулятор (только на Mac)

1. Установите Xcode из App Store
2. Запустите сервер: `npm start`
3. В терминале нажмите `i`

### Вариант 4: Android эмулятор

1. Установите Android Studio
2. Создайте виртуальное устройство в AVD Manager
3. Запустите эмулятор
4. Запустите сервер: `npm start`
5. В терминале нажмите `a`

## Первый запуск приложения

После открытия приложения:

1. Перейдите на вкладку **"Настройки"** (⚙️ внизу)
2. Нажмите **"Сгенерировать начальные данные (7 дней)"**
3. Включите **"Включить симуляцию"**
4. Выберите профиль пользователя (рекомендуется "Активный")

## Устранение проблем

### Приложение не открывается

- Убедитесь, что телефон и компьютер в одной Wi-Fi сети
- Попробуйте перезапустить Expo Go
- Проверьте, что порт 8081 не заблокирован файрволом

### Ошибки при загрузке

- Нажмите `r` в терминале для перезагрузки
- Нажмите `j` для открытия отладчика
