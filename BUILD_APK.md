# Инструкция по сборке APK файла

## Предварительные требования

1. Установлен Node.js (версия 16 или выше)
2. Установлен npm
3. Аккаунт Expo (бесплатный)

## Шаг 1: Установка EAS CLI

```bash
npm install -g eas-cli
```

## Шаг 2: Вход в аккаунт Expo

```bash
eas login
```

Создайте аккаунт, если у вас его еще нет (это бесплатно).

## Шаг 3: Настройка проекта для сборки

```bash
cd "/Users/pskripilev/Downloads/Мои файлы/healt"
eas build:configure
```

Эта команда создаст файл `eas.json` с конфигурацией сборки.

## Шаг 4: Сборка APK для Android

### Preview сборка (для тестирования)

```bash
eas build --platform android --profile preview
```

Эта команда:
- Создаст APK файл
- Загрузит его на серверы Expo
- Предоставит ссылку для скачивания

### Production сборка (для публикации)

```bash
eas build --platform android --profile production
```

## Шаг 5: Загрузка APK

После завершения сборки:

1. В терминале появится ссылка на скачивание
2. Или зайдите на https://expo.dev/accounts/[ваш-аккаунт]/builds
3. Скачайте APK файл

## Альтернативный способ: Локальная сборка

Если вы хотите собрать APK локально (требует Android Studio):

```bash
eas build --platform android --profile preview --local
```

## Настройка eas.json (опционально)

Если нужно настроить параметры сборки, отредактируйте файл `eas.json`:

```json
{
  "build": {
    "preview": {
      "android": {
        "buildType": "apk"
      }
    },
    "production": {
      "android": {
        "buildType": "apk"
      }
    }
  }
}
```

## Проверка статуса сборки

```bash
eas build:list
```

## Установка APK на устройство

1. Скачайте APK файл на Android устройство
2. Разрешите установку из неизвестных источников:
   - Настройки → Безопасность → Неизвестные источники
3. Откройте APK файл и установите

## Примечания

- Первая сборка может занять 10-15 минут
- APK файл будет около 30-50 МБ
- Необходимо подключение к интернету для сборки через EAS
- Бесплатный аккаунт Expo имеет лимиты на количество сборок в месяц

## Устранение проблем

### Ошибка при входе

```bash
eas logout
eas login
```

### Очистка кеша

```bash
eas build:configure --force
```

### Проверка конфигурации

```bash
eas build:configure
```

## Готово!

После успешной сборки у вас будет APK файл, который можно устанавливать на Android устройства.
